jQuery Version

Semantic-UI depends on jQuery to do DOM manipulation. If you're just using the styles of Semantic UI you can select none.

jQuery 2.x.x releases work with Internet Explorer 9 or later, and perform better than the 1.x.x releases. Pick a 1.x.x release if you need to support older browsers. Chrome, Firefox, Opera, and Safari have the same support with either.
